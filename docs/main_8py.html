Detailed Description

Runs a motor step response based on user inputs.

When run, this file asks the user for a setpoint and a proportional gain (K_p) value to do perform a motor step response with. The controller runs every 10ms and at each step the time and the encoder reading is recorded. After running, these values shall be transmitted through a serial port for plotting. See computerPlotting.py for details. Lastly, the program resets and asks for new step response parameters.
